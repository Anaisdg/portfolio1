Scary Author Identification

A basic natural language analysis with the aim of identifying three authors (Edgar Allan Poe, Marry Shelley, and H.P. Lovecraft) based on excerpts from their horror stories.



  • Built With:
  • • Python
  • • scikit-learn
  • • NLTK
  • • fastText
  • • kaggle dataset
  1. Methods Used for Predicting Authorship:
  2. • Word Frequency Classification
  3. • fastText Classification
  4. • Naive Bayes Classification



Figure 1. World size frequency for Edgar Allan Poe (EAP), Marry Wollstonecraft Shelley (MWS), and H.P. Lovecraft (HPL) from the training data set

Figure 2. World size frequency for Edgar Allan Poe (EAP), Marry Wollstonecraft Shelley (MWS), and H.P. Lovecraft (HPL) from the testing data set

Before building or using any classifiers the train (fig 1) and test (fig 2) data was compared. By visual inspection the comparison of the word size frequecy by author for the two data sets revealed that word size frequency could be used to help build a classifier. The classifier that my team and I built used only the word frequency to predic authorship. It outperformed Facebook's fastText classifier before any updates were made to customize the library for this specific analysis.

In additon to author predicitons, my team and I conducted several other analyses including: sentiment analysis, a POS (parts of speech) analysis, a pronoun analyis, sentence length, etc. We also examined the vocabulary size of the authors for the the train (fig 2) and test (fig 3) data.
